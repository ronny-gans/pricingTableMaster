# Pricing Table Master

A responsive pricing table designed using HTML and CSS. This project provides a clean and modern interface for displaying different pricing plans.

## Features

- Fully responsive design
- Supports multiple pricing tiers (Free, Team, Popular, Enterprise)
- Highlighted "Popular" plan for better visibility
- Uses Google Fonts (`Be Vietnam Pro`) for a modern look
- Custom styles and background images
- Hover effect on buttons for better user interaction

## Technologies Used

- **HTML** for the structure
- **CSS** for styling and responsiveness
- **Google Fonts** for typography
